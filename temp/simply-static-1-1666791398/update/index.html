    UPDATE

    Explorer 27 Kickstarter

    • 24/09/2022

    Explorer 27 is full throttled for a crowdfunding project on Kickstarter. When crowdfunding will be started is yet to be announced but intended to be live around November 2022. As series of unfortunate event is happened to the team since covid-19, the team is at limit that we won't be capable to resume any project if the situation isn't changed at the end of the year.

    Animated Desktop Wallpaper

    • 15/08/2022

    Starting project for animated wallpaper by Sasadara in collaboration with Ray Ryan. The project has no name yet, it is based on black and white painting style illustration that covering Wayang mythology as it's base. The project is intended to be created with around 6-8 figure, with Hanoman illustration is created as its base. The project is halted on 12 September 2022 as the computer is broken due to power surge. For the time being, animated wallpaper project collaboration is changed to different concept, with 3D artwork by Tyas Mahesta.

    Explorer 27R Released

    • 15/08/2022

    Explorer 27R is released as unlockable content on Rarible, with thousands of icon files intended for art collector only. The project begins to the next phase at the end of August that is intended for mass public consumption.

    Explorer 27 Resumed

    • 20/05/2022

    Ray Ryan continue foldico project and change the name to Explorer 27, the project is taking a shape of vector and color, it was focused on creating diversity of color variant with careful selection in design to make the design capable to contain as many variation as possible, concluded with 12 basic color and intended for NFT to be released on August 15 2022 to commerate 27 years of Windows Explorer.

    Explorer 27 Start Point

    • 14/01/2022

    Explorer 27 folder icon library is planned with Foldico as its name, the project is started with rough idea and digital sketch only by Ray Ryan, with roughly up to 2/5 hours, improving it bit by bit until March. On March it was taken down because of unfortunate problem, resumed in May.
